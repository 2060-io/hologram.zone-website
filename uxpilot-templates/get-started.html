Quick Deployment

Deploy with Docker in minutes and start serving verifiable content immediately

1

Get VS Agent

Choose your preferred installation method: Docker or local setup

Docker (Recommended)

Fastest way to get started

docker pull io2060/vs-agent:dev

Pre-configured and ready to run with minimal setup required

Local Installation

For development and customization

Clone and run locally for full control:

VS Agent Repository

Refer to repository for detailed local installation instructions

2

Configure Environment Variables

Set up essential configuration for your Verifiable Service

Required Environment Variables

AGENT_PUBLIC_DID

Required

Your service's public decentralized identifier (DID). Hologram supports did:webvh Method, a web-based identifier.

Format:

did:webvh:mypublichost.com

You'll need a public endpoint for your agent. Learn more about did:webvh Method

AGENT_LABEL

Required

String displayed in invitations to identify your service

Example:

"My First Hologram VS"

AGENT_INVITATION_IMAGE_URL

Recommended

URL of a logo shown in invitations alongside the label

Example:

https://hologram.zone/images/ico-hologram.png

Important Configuration Note

The public endpoint must be reachable at the same host you've set up in your agent's public DID. Ensure your hosting configuration matches your DID specification.

3

Run Your Hologram Verifiable Service

Deploy your VS Agent container with the configured environment variables

Docker Run Command

docker run -p 80:3001 -p 3000:3000 \ -e AGENT_PUBLIC_DID=did:webvh:myhost.com \ -e AGENT_LABEL="My First Hologram VS" \ -e AGENT_INVITATION_IMAGE_URL=https://hologram.zone/images/ico-hologram.png \ --name io2060/vs-agent:dev

Port Configuration

3001

Public Port

User-facing endpoint for service interactions

3000

Admin API Port

Private port for administrative operations

Security Best Practice

Ensure the public endpoint is reachable at the same host set in your agent's public DID.

Example: If DID is did:webvh:myhost.com, your service must be accessible at myhost.com

4
